Typical new server setup - Linode/CentOS
========================================

```
shh root@server

yum update

sudo hostnamectl set-hostname <name>

useradd -d /home/dtron -m -G wheel dtron

passwd dtron

vim /etc/ssh/sshd_config  # AllowRootLogin no

yum install yum-cron zsh git

reboot
```
